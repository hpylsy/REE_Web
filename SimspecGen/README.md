# 全自动光谱生成程序

## 一、使用方法
1. 进入项目目录下的 `saved` 文件夹。
2. 打开并修改 `contents.csv` 文件。
3. `contents.csv` 必须包含以下字段：
   - 编号
   - 基体元素的含量
4. 保存文件后，运行程序进行光谱生成。

## 二、注意事项
1. 请确保 `contents.csv` 的表头命名规范、数据格式正确。
2. 缺少“编号”或“基体元素的含量”字段会导致程序无法正常处理数据。
3. 建议先备份原始 `contents.csv`，再进行批量修改。
4. 如果需要更改基体元素种类或稀土元素种类（含量）则需要在Gen.py程序中手动更改
